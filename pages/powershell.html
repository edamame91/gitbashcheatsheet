PowerShell Cheat Sheet

Navigation

Get-Location Show current working directory
Set-Location C:\\path\\to\\dir Change directory (alias: cd)
Get-ChildItem -Force List files including hidden items (alias: ls)
Push-Location .. Save current location and move
Pop-Location Return to previous saved location

Files & Folders

New-Item notes.txt -ItemType File Create a file
New-Item logs -ItemType Directory Create a directory
Copy-Item src.txt backup.txt Copy file or folder
Move-Item old.txt archive\\old.txt Move or rename item
Remove-Item temp.txt Delete file
Remove-Item .\\build -Recurse -Force Delete directory recursively destructive

Read & Search

Get-Content app.log Print file content
Get-Content app.log -Tail 50 -Wait Follow live logs (tail -f equivalent)
Select-String -Path *.log -Pattern "ERROR" Search for text in files
Get-ChildItem -Recurse -File | Select-String "TODO" Recursive content search

Pipeline Basics

Get-Process | Sort-Object -Property CPU -Descending Sort processes by CPU usage
Get-Service | Where-Object {$_.Status -eq "Running"} Filter only running services
Get-ChildItem | Select-Object Name, Length, LastWriteTime Project selected properties
Get-ChildItem | Measure-Object Count and summarize piped objects

Process & System

Get-Process List running processes
Stop-Process -Name notepad -Force Kill process by name
Get-Service List services
Restart-Service -Name spooler Restart a service
Get-ComputerInfo Show machine and OS details

Scripting Essentials

$name = "Alex" Create variable
Write-Host "Hello $name" Print formatted output
if ($LASTEXITCODE -ne 0) { exit 1 } Basic conditional on previous command
foreach ($f in Get-ChildItem *.txt) { $f.Name } Loop through matching files
.\\script.ps1 -WhatIf Run script in safe preview mode

Everyday Useful One-Liners

Get-Command *item* Find commands by wildcard name
Get-Help Get-ChildItem -Examples Show practical examples for a cmdlet
Get-NetIPAddress | Where-Object {$_.AddressFamily -eq "IPv4"} List IPv4 addresses
Test-Connection github.com -Count 2 Quick network connectivity check
Get-History | Select-Object -Last 20 Show your recent commands
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser Allow running local scripts security-sensitive
